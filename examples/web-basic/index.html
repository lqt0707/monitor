🚀 Monitor SDK

Web 端监控SDK示例集合

📋 使用说明

1. 启动后端服务： npm run server

2. 安装依赖： npm install (在 examples/web-basic 目录)

3. 启动开发服务器： npm run dev

4. 选择下面的示例进行测试
